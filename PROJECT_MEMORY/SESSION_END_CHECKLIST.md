# 会话结束前自检（任意 AI / IDE）

复制本清单；离开电脑前尽量勾完（可与用户约定「仅重大变更必填」）。

- [ ] **`NEXT_ACTION.md`**（或你们约定的「下一步」文件）：下一步 / 阻塞  
- [ ] **`PROJECT_MEMORY/INDEX.md`**：**若使用本目录**，里程碑可在登记簿 **追加一行**（可选）；详细叙事放在 `iterations/` 或 PR / Wiki    
- [ ] **`CURRENT_STATUS.md`**：进度 % / 阶段 / 构建与测试结论  
- [ ] **`TASK_QUEUE.md`**：已完成 / 进行中 / 待办优先级  
- [ ] **`SESSION_LOG.md`**：本次改动文件、测试结果、关键决策一句话  
- [ ] **`DECISIONS.md`**：若有架构级取舍已更新  
- [ ] **较大里程碑**：在 **`PROJECT_MEMORY/iterations/`** 新增或追加一篇归档（含 commit hash）  
- [ ] **`git status`**：若项目使用 Git 且约定提交记忆，则按需 **commit** / **push**  

**勿写入**：密钥、token、个人隐私全文。
